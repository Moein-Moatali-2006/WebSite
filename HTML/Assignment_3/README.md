# My Blog Website

This project is a simple HTML-based personal blog created by **Moein
Moatali**.

## Features

-   Navigation bar with links

-   Embedded video player

-   YouTube and Aparat iframe players

-   Login form with captcha

-   Chess board made using HTML table

-   Google Maps iframe

-   Audio players

-   Footer section


## How to Run

Just open the HTML file in any browser.
